# Firebase Demo

# Technologies

- `Vanilla JavaScript`
- `Firebase`
- `Webpack`

## Run this demo project

```
// Add yours

const firebaseConfig = {
  apiKey: "your_api_key",
  authDomain: "your_auth_domain",
  projectId: "your_project_Id",
  storageBucket: "your_storage_bucket",
  messagingSenderId: "your_messaging_sender_id",
  appId: "your_app_Id",
}
```

```
Install dependencies:
npm install

Run dev server: 3000
npm run dev

Build prod:
npm run build
```
